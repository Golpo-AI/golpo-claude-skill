# Background music (`--bg_music`)

Optional. Adds a music bed under the narration. One value per video.

| Value | Mood |
|---|---|
| `jazz` | Mellow, smooth, late-night feel. |
| `lofi` | Chill, study-with-me, ambient beats. |
| `whimsical` | Light, playful, kid-friendly. |
| `dramatic` | Cinematic, high-stakes. |
| `engaging` | Upbeat, energetic, "promo" feel. |
| `hyper` | Fast, intense, action / sports. |
| `inspirational` | Uplifting, motivational. |
| `documentary` | Reflective, "this American life" feel. |

Omit `--bg_music` for narration-only audio (no music bed).
